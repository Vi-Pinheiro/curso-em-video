Outras formatações

Código fonte

Aplicar uma formatação de código, podemos usar a tag PRE e a tag CODE.

        
            A Tag PRE mantém a formatação utilizada no código fonte.
            Já a Tag CODE mantém as letras monoespaçadas representando as características de um código.

Citações completas

Neste caso a Tag aplicada se chama blockquote. Ela é utilizada para aplicar uma citação completa, aplicando uma formatação própria para o trecho.

Este é um trcho de exemplo, da tag citada. Dentro dele pode ser aplicada a propriedade cite para colocar a origemm da citação.
